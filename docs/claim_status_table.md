## Recommended Use Rules

- `title / abstract`: only `confirmed` or `supported but scoped` claims with explicit scope qualifiers
- `Results`: `confirmed` claims freely; `supported but scoped` claims with clear caveats
- `Discussion only`: all `still conjectural` claims, plus broader speculative extrapolations of supported claims

## One-Line Editorial Rule

If a sentence would be invalid after adding the phrase `across the tested family and within current bookkeeping`, then it is not yet strong enough for title or abstract use.
