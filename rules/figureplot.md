Global style rules for all Extended Data figures:

- Produce publication-ready scientific figures, not conceptual illustrations.
- White background, clean vector-first layout, minimal clutter.
- Use the same notation as the main figures: Pi_m, Pi_f, Pi_U, tau_g, l_g.
- Use the same canonical labels consistently:
  OP_SUCCESS_TIP, OP_EFFICIENCY_TIP, OP_SPEED_TIP, OP_BALANCED_RIDGE_MID, OP_STALE_CONTROL_OFF_RIDGE.
- Preserve the finalized claim hierarchy:
  main figures carry the ceiling claim,
  extended data provides robustness, chronology, audit, and detail.
- Do not promote out-of-scope claims:
  no full crossing-completion law,
  no unrestricted universality,
  no coefficient-exact collapse,
  no full thermodynamic closure.
- Keep typography compact and readable.
- Every panel should have a short takeaway title.
- Export final figures as high-resolution PNG and, where possible, SVG or PDF.
- Write a short panel manifest documenting which source files generated each panel.
